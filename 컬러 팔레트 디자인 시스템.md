## 컬러 사용 가이드

### Primary Green (`#00704A`)
- **용도**: 브랜드 정체성을 나타내는 핵심 색상
- **적용 위치**: 로고, 주요 CTA 버튼, 링크, 아이콘
- **주의사항**: 전체 면적의 10~15%로 절제하여 사용 시 고급스러운 인상 유지

### Dark Green (`#1E3932`)
- **용도**: 깊이감과 안정감을 주는 보조 색상
- **적용 위치**: 푸터, 하단 정보 섹션, 헤더 텍스트, 다크 배경 영역
- **주의사항**: 반드시 밝은 텍스트(`#FFFFFF` 또는 `#F7E6C4`)와 함께 사용

### Pink/Red (`#D4145A`)
- **용도**: 시선을 끄는 강조 색상 (시즌/프로모션)
- **적용 위치**: 히어로 배너, 할인/이벤트 알림, 강조 배지
- **주의사항**: 넓은 면적에 사용 시 흰색 텍스트로 가독성 확보. 과도한 사용 자제

### Cream/Gold (`#F7E6C4`)
- **용도**: 프리미엄 느낌의 장식 색상
- **적용 위치**: 캘리그라피, 장식 텍스트, 미묘한 배경 악센트
- **주의사항**: 소량으로 포인트 역할. 어두운 배경 위에서 효과적

### White (`#FFFFFF`)
- **용도**: 깔끔함과 여백을 제공
- **적용 위치**: 메인 콘텐츠 배경, 네비게이션, 카드 컴포넌트
- **주의사항**: 충분한 여백(padding/margin)과 함께 사용하여 시각적 호흡감 확보

## 반응형 고려사항

### 다크모드 대응

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* Background 반전 */
    --color-bg-primary: #1E3932;
    --color-bg-secondary: #0D1F1A;
    --color-bg-banner: #A8103E;

    /* Text 반전 */
    --color-text-primary: #F0F0F0;
    --color-text-secondary: #D4D4D4;
    --color-text-muted: #999999;

    /* Interactive 조정 */
    --color-button-primary: #00A862;
    --color-button-hover: #00C472;
    --color-link: #00A862;
    --color-link-hover: #00C472;

    /* Border 조정 */
    --color-border-light: #3A5A50;
  }
}
```

### 화면 크기별 색상 적용

```css
/* 모바일 (768px 이하) */
@media (max-width: 768px) {
  /* 배너 영역 단순화 - 그라데이션 줄이기 */
  .hero-banner {
    background: var(--color-bg-banner);
  }

  /* 텍스트 대비 강화 - 작은 화면에서 가독성 확보 */
  .nav-text {
    color: var(--color-text-primary);
    font-weight: 600;
  }
}

/* 태블릿 (769px ~ 1024px) */
@media (min-width: 769px) and (max-width: 1024px) {
  .hero-banner {
    background: linear-gradient(135deg, var(--color-bg-banner), #E83670);
  }
}

/* 데스크톱 (1025px 이상) */
@media (min-width: 1025px) {
  .hero-banner {
    background: linear-gradient(135deg, var(--color-bg-banner), #E83670, #F0507A);
  }
}
```
